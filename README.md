# GMail Word Cloud

The key different about this code is the graph is generated using emails from specific emails.

## Installation

```
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
python -m nltk.downloader punkt stopwords
```

## Usage

```
./gmail_word_cloud.py
```
